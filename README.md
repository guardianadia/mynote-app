
# MyNote App

A full-stack notes application built using Flutter and Supabase.

## Features
- User authentication (login & register)
- Password reset via email
- Username recovery system
- Online storage using Supabase
- Offline storage using Hive
- Note pinning and favorites
- AI-powered summary feature

## Tech Stack
- Flutter
- Supabase
- Hive (local storage)

## How to Run
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
